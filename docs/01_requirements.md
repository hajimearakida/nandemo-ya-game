## コアゲームループ

```
[毎日の朝] 残り日数・資金・評判を確認
  → 依頼ボードから依頼を選ぶ（複数掲示）
  → 住民と会話 → ミニゲーム実行
  → 成功: 報酬獲得・関係値UP
    失敗: 評判DOWN（一定以下で廃業エンド）
  → 報酬でお店を強化・拡張
  → 翌日へ
```

## 主要システム

### 依頼システム
- 毎日複数の依頼が依頼ボードに掲示される（ランダム＋固定イベント）
- 依頼ごとに難易度・報酬・依頼人が設定される
- 断ることも可能（評判値が微減）
- 依頼をこなすことで住民の関係値が上がり、隠し依頼が解放される

### 経済システム
- 依頼完了で報酬（ゲーム内通貨）を獲得
- お店の改装・道具の購入でミニゲーム難易度が下がったり新依頼が解放される
- 資金不足＋評判低下 → 廃業エンド

### 時間システム
- ゲームは60日間の制限あり
- 日数終了時の評判・資金・実績によってエンディングが分岐

### 評判・関係値
- 街全体の「評判値」と住民個別の「関係値」の2軸
- 評判が一定以下 → 廃業エンド（ゲームオーバー）
- 関係値が高い住民 → 隠し依頼・特別会話が解放

### セーブシステム
- オートセーブ＋手動セーブ（JSON形式でローカル保存）

## Steam 実績

| 実績名 | 条件 |
|--------|------|
| みんなの友達 | 全住民と関係値MAX |
| 廃業寸前 | 評判値が底を打って復活 |
| スピードマスター | 依頼を最速でクリア |
| 奇想天外 | 特殊な方法で依頼を解決 |
| 隠しの依頼人 | 隠し依頼人を発見 |
| 完璧な最終日 | 最終日に全住民と関係値MAX |
