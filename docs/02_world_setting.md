# 世界観・ストーリー設定

## 舞台

**ゴタマゼ町**（仮称）

人間・獣人・小柄な妖精族・元魔法使いの老人・喋る動物……色々な種族が特に理由もなく混ざって暮らしている、のどかで少し不思議な町。大きな事件も英雄も関係ない。ただ毎日が続いている。

## 主人公

| 項目 | 内容 |
|------|------|
| デフォルト名 | ハコ（変更可）|
| 種族 | 人間（旅人）|
| 年齢 | 20代前半くらい |
| 動機 | 旅の途中でこの町に立ち寄り、廃業寸前の「なんでも屋」を「まあいいか」で引き継いだ |

> 「前の店主はどこ行ったんですか？」
> 「さあ。ある日いなくなって、鍵だけ郵便受けに入ってたんですよ」
> 「……で、なんで俺（私）が？」
> 「あなたが一番近くにいたので」

主人公に壮大な夢はない。依頼をこなすうちに住民との関係が生まれ、いつの間にかこの町が「居場所」になっていく。

## 60日間の大きな流れ

| 期間 | 出来事 |
|------|--------|
| 1〜15日目 | 町に慣れる。基本的な依頼をこなし、住民と顔見知りになる |
| 16〜35日目 | 住民たちの事情が少しずつ見えてくる。関係値が高い住民の隠し依頼が解放される |
| 36〜50日目 | 町全体に関わる小さな「出来事」が起きる（祭り・揉め事・謎の依頼人の出現など）|
| 51〜60日目 | ラストスパート。評判・関係値・資金でエンディングが分岐 |

## エンディング分岐

| エンド名 | 条件 |
|----------|------|
| 「ここが家」エンド | 全住民と関係MAX・評判高い → 旅をやめて町に住み着く |
| 「また旅へ」エンド | 評判・資金クリア・関係値は中程度 → 旅を再開するが「いつか戻る」 |
| 「廃業」エンド | 評判・資金が底をつく → なんでも屋を畳む（BAD）|
| 「謎の依頼人」エンド | 隠し依頼人ルート（タマ＋ロッカ）を完走 → 特殊ED |
| 「完璧な便利屋」エンド | 全依頼クリア・全実績 → 隠しEND |

## ほのぼのファンタジーらしい依頼のトーン

- 「ドラゴンが庭に居着いて困っている（追い払うか、仲良くするか）」
- 「魔法陣が消えなくて床が拭けない」
- 「元勇者の老人が若い頃の剣を探している」
- 「妖精族の子が人間の言葉で手紙を書きたい」
- 「喋る猫が遺言状を書きたいと言っている」
