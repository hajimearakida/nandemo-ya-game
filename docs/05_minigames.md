# ミニゲーム仕様

## ミニゲーム5種の定義

| 種類 | 内容 | 操作感 |
|------|------|--------|
| **パズル型** | 物を整理・修理・組み立てる | ドラッグ＆ドロップ、並び替え |
| **タイミング型** | 決まったタイミングでボタン/クリック | リズム・ゲージ調整 |
| **選択肢型** | 会話・交渉・仲裁 | テキスト選択肢（複数回）|
| **探索型** | 画面内から物/人を見つける | クリックで調べる |
| **特殊型** | 隠し依頼専用の一回限りのゲーム | 依頼ごとに異なる |

---

## 依頼難易度とミニゲームの関係

```
難易度★1  →  選択肢型・探索型（失敗しにくい）
難易度★2  →  タイミング型・パズル型（練習が要る）
難易度★3  →  複合型（探索＋選択肢など2段階）
隠し依頼   →  特殊型（一発勝負・高報酬）
```

## ショップ強化との連携

| 購入アイテム | 効果 |
|------------|------|
| 道具箱 | パズル型の制限時間が延びる |
| 地図 | 探索型でヒントが増える |
| 話術の本 | 選択肢型で選択肢が1つ増える |
| 丈夫な鞄 | タイミング型の判定が緩くなる |

## Unityクラス設計

```csharp
public abstract class BaseMinigame : MonoBehaviour
{
    public abstract void StartMinigame(QuestData quest);
    public event Action<bool> OnMinigameCompleted; // true=成功
}
```

| クラス名 | 種別 |
|----------|------|
| PuzzleMinigame | パズル型 |
| TimingMinigame | タイミング型 |
| ChoiceMinigame | 選択肢型 |
| SearchMinigame | 探索型 |
| SpecialMinigame | 特殊型 |
